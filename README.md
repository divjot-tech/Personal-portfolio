# 🌐 Personal Portfolio

A modern, responsive, and animated personal portfolio built with **Next.js**, **TypeScript**, **Tailwind CSS**, and **Framer Motion** to showcase my projects, technical skills, certifications, and academic journey.

🔗 **Live Demo:** https://personal-portfolio-xi-one-47.vercel.app/

---

## ✨ Features

- 🎨 Modern and clean UI/UX
- 📱 Fully responsive across all devices
- ⚡ Built with Next.js 16 App Router
- 🎭 Smooth animations using Framer Motion
- 🌟 Interactive hero section
- 🖱️ Mouse glow effect
- 📊 Scroll progress indicator
- 🚀 Animated page loader
- 💼 Professional projects showcase
- 🏆 Certifications section
- 🎓 Education timeline
- 🛠️ Skills categorized by technology
- 📬 Contact section with social links
- 🌙 Optimized for performance and SEO

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/divjot-tech/Personal-portfolio.git
```

### Navigate to the project

```bash
cd Personal-portfolio
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:3000
```
